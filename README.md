# Delivery Deadline Calculator

Calculadora de prazo de entrega via terminal, desenvolvida em Python.

## Funcionalidades
- Entrada de produto, quantidade e dias úteis
- Cálculo automático da data de entrega
- Saída formatada no terminal

## Como usar
```bash
python main.py
```

## Tecnologias
- Python 3
- Módulo datetime (nativo)
